# Hugo
- Installieren Sie Hugo mit folgendem Befehl.
```bash
sudo snap install hugo
```
- Erstellen Sie ein neues Projekt mit folgendem Befehl.
```bash
hugo new site Hugo
```
- Wechseln Sie in das Verzeichnis mit folgendem Befehl.
```bash
cd Hugo
```
